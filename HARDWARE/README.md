# HARDWARE 文档索引

本目录包含工程的硬件驱动、接口头文件以及两份配套说明文档。

## 文档列表

- README_API.md
    面向函数调用，详细说明 ADS1118、AD7616、delay、LCD、AD9834 的接口、参数、返回值与示例。

- README_PORTING.md
    面向移植与改板，详细说明 GPIO 映射、初始化顺序、时钟宏、常见故障定位方法。

## 目录说明

- Inc/
    对外头文件，包含接口声明、宏配置与双语说明。

- Src/
    驱动源文件，包含协议实现与底层硬件访问逻辑。

## 注释风格

本次整理后，HARDWARE 目录的注释统一采用中英文双语：

- 中文负责说明用途、场景和注意事项。
- English 用于补充术语、接口含义和移植时的对应概念。

## 阅读建议

如果你是写业务逻辑：

1. 先看 README_API.md
2. 再看对应的头文件注释
3. 涉及 AD7616 时，再结合并口/串口模式确认初始化顺序与读取流程

如果你是换板子或改引脚：

1. 先看 README_PORTING.md
2. 再检查 Core/Inc/main.h 与 HARDWARE/Inc/*.h 映射关系
3. AD7616 额外检查 DB0~DB15 是否跨端口、BUSY 是否可读、DWT 延时是否正常
